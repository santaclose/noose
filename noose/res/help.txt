Use the middle mouse button to pan the view.

Use the left mouse button to move nodes around by
dragging their top bars.

Use the left mouse button to change node input values.
Numeric values are modified by dragging across the
horizontal axis.

Use the left mouse button to connect two nodes by clicking
over a pin and dragging to another node's pin.

Use the right mouse button to select nodes by clicking on
their top bar.

Use the right mouse button to delete connections by clicking
and dragging over them.

Use the right mouse button to enter numeric values using the
keyboard. Press the enter key after introducing the value.

Use the delete or backspace key after selecting a node to
delete it.

Use the spacebar to find nodes by typing it's name.
